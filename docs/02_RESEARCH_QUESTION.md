# Research question, objectives & study design

## Central research question

> Across the datasets generated by different labs regarding independent poxvirus transcriptome, which of the host genes and further which of the host pathways are the most reproducibly remodeled by infection? Which host hubs are targeted the most consistently?

The research question of this project is, in a deliberate way, genome-wide and family-agonistic, by which we mean that it does not pre-commit to any one family of proteins. Remodeling of host translation/RNA-helicase is being examined here as one pre-specified sub-question within this broader map and not as the primary headline. This remodeling of host translation/RNA-helicase is of standing interest because the literature surrounding poxvirus-helicase is so sparse here. This breadth that is being described here is because of the guidance that I got from my project mentor at the beggining where the advice was: do not pre-narrow to a single protein family. 

## Sub-questions and hypotheses

1. **Reproducibility (primary).** Are there host genes where the direction of change that they have remains consistent across multiple different independent studies (different labs, viruses, cell types)?  *H1: there is a substantial coherent set of host genes that are remodeled reproducibly.*
2. **Program structure.** Which biological programs are being over-represented throughout the reproducible genes? What direction? *H2: host shutoff (translation that is down-regulated/ribosomal and cell-cycle programs) and immune/inflammatory induction are being dominant.*
3. **Targeted hubs.** When they are mapped onto a network, which host genes are of the most central and of the most coordinately targeted? *H3: the reponse in anchored by a small set of hub genes.*
4. **RNA-helicase sub-question.** In the broad program are RNA helicases/translation factors being remodeled coherently? Is any single helicase a defensible candidate for follow-up? *H4: helicases form a minor but consistent*
  module; one or two at the most are robust enough to prioritize.*
5. **Robustness & validity.** Does the signature survive leaving out any one study? Transfer to an unseen study? Appear in an independent modality (protein) and cell context? *H5: yes for the core shutoff program.*

## Objectives

1. Systematically discover and the systematically tier public poxvirus host-response datasets.
2. Then I am going to harmonize differential expression across/between them and this will be done with one consistent pipeline.
3. Run a genome-wide meta-analysis in search for random-effects with leave-one-study-out
  robustness which is for validation and then later continue onto heterogeneity classification.
4. Quantify the over-representation of host-program and also rank the host hubs being targeted and rank them via
  network analysis — both family-agnostic.
5. Stress-test portability (leave-one-dataset-out classifier). Validate
  externally (independent study; single-cell + proteomic modalities).
6. Report a ranking of transparent candidate that is also falsifiable for experimental follow-up for future studies to observe.

raw counts; one harmonized from author DESeq2 tables.
- **Tier B (external validation):** GSE185520 (VACV D10 shutoff).
- **Tier C (context only):** long-read *viral* transcriptome studies and designs that are non-harmonizable.
- **External validation (independent):** Matía et al. 2024 single-cell + proteomics.

**Methods overview:** This is going to be the plan that I am going to be working toward so that there is consistency and also putting it here for my own organization and for even future reproducibility: PyDESeq2 differential expression on harmonized counts → DerSimonian–Laird random-effects meta-analysis (Q/I²/τ², BH-FDR, LOSO) → hypergeometric host-program enrichment → cross-study co-expression network hub ranking → leave-one-dataset-out portability check → independent external validation. The layer of vaccinia-infection-only sensitivity allows for a clear seperation between the true infection biology and the effector-contrast effects.

## Success criteria

- A ranked and further a reproducible host-gene set that is recoverable across ≥3 independent studies.
- A clearly interpretable set of over-represented programs (that also have the direction).
- At least one externally and cross-modally supported claim (e.g. host shutoff).
- An honest hierarchy for any prioritized gene (for example helicase candidates). This also with heterogeneity and the associated limitations being said and stated.

## Scope and limitations

This project I am aiming it to be computational and for this to be **hypothesis-generating**. All of the primary signals are on the transcript level. The evidence for protein/ribosome profiling is going to be used when and where it avaliable but the key point is that it will not be assumed. The cross-species ortholog mapping (Vero) and also the public-metadata noise are cavets here. The pooled meta that is present causes a mixing of the true infection with the effector contrasts (as was mentioned above also) which is now being handeled by a sensitivity analysis (vaccina-only) (see the "Study design" section above also for more info about this). There is not and will not be any causal or mechanistic claims within this project and there is no claim that this is the primary discovery of the underlying datasets.
